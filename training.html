Curriculum

12-Week Learning Journey

Our structured curriculum takes you from beginner to professional web designer with hands-on projects and real-world applications.

Week 1-2

HTML & CSS Fundamentals

Learn the building blocks of web design. Master semantic HTML, CSS selectors, flexbox, and grid layouts.

Week 3-4

Responsive Design & Mobile-First

Create websites that adapt to any screen size. Learn media queries, breakpoints, and mobile-first design methodology.

Week 5-6

JavaScript & Interactivity

Add dynamic behavior to your websites. Master DOM manipulation, events, and modern JavaScript ES6+ features.

Week 7-8

CSS Frameworks & Tailwind

Speed up development with CSS frameworks. Deep dive into Tailwind CSS and utility-first design principles.

Week 9-10

React & Modern Frontend

Build dynamic web applications with React. Learn components, hooks, state management, and modern development workflows.

Week 11-12

Portfolio & Career Prep

Create a professional portfolio website and prepare for job interviews. Learn deployment, version control with Git, and industry best practices.
